📝 Challenge: User Input
Create a program that asks the user for their name and greets them!

Task:
  1. Ask the user "What is your name?"
  2. Store their answer in a variable called "name"
  3. Print a greeting message: "Hello, [name]!"
💡 Example Output (if user enters "Alex"):
Hello, Alex!
💡 Hint: Use the input() block to get user input!
You can combine strings using the join block or + operator.
