📝 Challenge: Type Conversion
Create a program that converts a string number to an integer and then back to a string!

Task:
  1. Create a variable with the string value "42"
  2. Convert it to an integer using int()
  3. Then convert it back to a string using str()
  4. Print the final result
💡 Example Output:
42
💡 Hint: Use the int() and str() blocks from the Type Conversion category!
The int() block converts text to a number, and str() converts numbers back to text.
